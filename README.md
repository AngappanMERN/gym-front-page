# Arttavive — Single Page Sample

This workspace contains a single-page site sample for "Arttavive".

Files:
- `gym webpage.html` — main HTML file (open in browser)
- `gym webpage.css` — stylesheet for the page

To view locally, open `gym webpage.html` in your browser (double-click or use a local server).

Optional: run a quick static server (Python 3):

```bash
python -m http.server 8000
# then open http://localhost:8000/gym%20webpage.html
```
